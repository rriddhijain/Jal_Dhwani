Vision

The Agentic Brain
Edge: ESP32-S3 performs real-time FFT frequency analysis to detect pump health locally.

Cloud: AWS Bedrock (Claude 3.5 Sonnet) synthesizes pump data with live weather forecasts to automate irrigation schedules.

Action: Amazon Polly triggers voice calls to farmers in local languages explaining pump state.
